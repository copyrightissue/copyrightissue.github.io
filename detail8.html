Affirmations App - Displaying Affirmations Using Kotlin

This project was a deep dive into Kotlin development, aiming to create an affirmations app that showcases scrolling strings as a scrollable list.

The Affirmations app was developed as a part of the Kotlin Basics App Program, providing users with a simple yet uplifting experience as they scroll through various affirmations.

  1. Introduction to Kotlin's basics for app development.
  2. Creation of a scrollable list interface.

To explore the code, visit the project on GitHub.

Key Learnings

Developing the Affirmations app provided insights into:

  • Handling scroll events in Kotlin.
  • Designing a user-friendly UI for app users.
