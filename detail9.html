App Development 391

Chitter - Birdwatching Adventure

Chitter is a birdwatching app where users can identify, record, and earn points for spotting different bird species.

Description of Chitter app features such as LiveData points tracking, navigation with fragments, view model, and search functionality for bird species.

  1. Live tracking of points for bird sightings.
  2. Navigation with fragments.
  3. Search feature to quickly find birds in the list.
  4. Details page for descriptions of birds and their point value.

My deliverable can be found on GitHub.

Key Takeaways

Throughout this project, I acquired a deeper understanding of Android development, particularly in building intuitive user interfaces with Kotlin. The integration of theme and material design added a layer of sophistication to the app, making it user-friendly and visually appealing.

  • Practical implementation of live data.
  • Understanding and applying material design guidelines for Android apps.
  • Using fragments for navigation.
  • Using filter and searchView to implement rudimentary search function.
