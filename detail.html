Web Design 181 Travel Blog

The assignment for Web Design was to create a travel blog. We had free range for styling and theme, and had to create several pages and links to demonstrate knowledge.

The original assignment was to create a travel blog with at least 3 layers of links. Then, link to each of those pages and practice linking to outside websites. We also had to insert images and come up with a central cohesive theme.

  1. Learned how to do linking with anchor tags.
  2. Learned CSS tricks for making a Navigation bar.

My deliverable can be found here.

What I learned

This project taught me a lot of CSS tricks that I used to get the navigation bar at the top. The permanent links look like buttons, and the current page is highlighted. I also learned how to insert emojis as favicons. The assignment itself taught me how to use links and build a multi-layered website.
